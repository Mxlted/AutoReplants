# AutoReplant

A client-side Fabric mod that automatically replants crops when you harvest them while holding a hoe.

## Features

- Automatically replants crops when broken with any hoe (wooden, stone, iron, golden, diamond, netherite)
- Works with wheat, carrots, potatoes, beetroots, nether wart, torchflower, and pitcher crops
- Seeds must be in your hotbar to replant
- Works at any crop growth stage
- Client-side only, works on any server

## Version Compatibility

- Minecraft: 26.1.2
- Fabric Loader: 0.18.4 or higher
- Fabric API: 0.147.0+26.1.2

## Build from Source

```bash
git clone https://github.com/Mxlted/autoreplant.git
cd autoreplant
gradlew build
```
